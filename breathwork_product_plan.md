# Breathwork Mini-App — Product Plan v1.0

## 1. Продуктовое позиционирование

**Одной строкой:** Breathwork-инструмент для IT-перформеров, которые управляют состоянием через тело — без эзотерики, но с глубиной.

**Отстройка от конкурентов:**

| | Calm / Headspace | BadBuddhas | Твой продукт |
|---|---|---|---|
| Тон | Мягкий, mass-market | Комьюнити, мистика + наука | Прямой, функциональный, с глубиной |
| Формат | Библиотека медитаций | Платформа тичеров | Авторский breathwork-инструмент |
| Аудитория | Все | «Духовные предприниматели» | IT-перформеры под нагрузкой |
| Визуал | Пастельный, иллюстрации | Дерзкий, контркультурный | Тёмный минимализм + 3D-арт (см. раздел 7) |
| Ощущение | «Расслабься» | «Ты — проводник» | «Верни себе контроль» |

**Ключевое обещание продукта:**
«Breathwork, который работает как инструмент — быстро, системно, с ощутимым результатом.»

## 2. Целевая аудитория — JTBD-карта

### Архетип: «Рациональный мистик»
IT-специалисты 25-38 лет с высокой когнитивной нагрузкой. Трекают HRV, но чувствуют энергии. Хотят научную базу, но не сухую нейронауку. Уже соприкасались с медитацией, йогой, терапией. Ценят системность, эстетику и глубину одновременно.

### Job 1 — «Перезагрузка под нагрузкой» (ВХОД в продукт)

- **Ситуация:** Вечер после 12-часового дня. Тело в напряжении, голова гудит. Завтра снова на максимум.
- **Мотивация:** Быстро сбросить напряжение и восстановить ясность, не тратя час на практику.
- **Ожидаемый результат:** Через 10-15 минут — ощутимое изменение состояния. «Я чувствую, что перезагрузился.»
- **Конкурирующие решения:** Алкоголь, сериалы, баня, спорт, CBD.
- **Критерии выбора:** Быстро (≤15 мин), работает с первого раза, не требует подготовки, можно делать дома.

**→ Продуктовый ответ:** Короткие сессии 7-15 мин с конкретным функциональным результатом. Категория «Быстрая перезагрузка». Минимальный онбординг — открыл, выбрал, дышишь.

### Job 2 — «Почувствовать глубину» (УДЕРЖАНИЕ + дифференциация)

- **Ситуация:** Всё вроде есть — деньги, проекты, признание — но внутри пустота. Живу на автопилоте. Хочу почувствовать что-то настоящее.
- **Мотивация:** Вернуть контакт с собой, прожить эмоции, которые подавлял. Опыт, а не информация.
- **Ожидаемый результат:** «Я что-то почувствовал. Это было мощно. Хочу ещё.»
- **Конкурирующие решения:** Церемонии, ретриты, психотерапия, экстремальный спорт.
- **Критерии выбора:** Безопасно, направлено, с проводником (голосом), создаёт пространство для переживания.

**→ Продуктовый ответ:** Длинные иммерсивные сессии 25-45 мин с музыкой и голосовым сопровождением. Категория «Глубокие практики». Контент-курсы по 7-14 сессий с прогрессией.

### Job 3 — «Выстроить систему» (КОНВЕРСИЯ в постоянного пользователя)

- **Ситуация:** Попробовал breathwork, почувствовал эффект. Хочу сделать это привычкой, но не знаю как встроить в расписание и как прогрессировать.
- **Мотивация:** Системность, видимый прогресс, ощущение роста — как в тренировках или трекинге привычек.
- **Ожидаемый результат:** «У меня есть моя практика. Я вижу прогресс. Это часть моей рутины.»
- **Конкурирующие решения:** Тренер/коуч, фитнес-приложения (Strava-логика), дневники, таймеры привычек.
- **Критерии выбора:** Трекинг, прогрессия, рекомендации следующего шага, ощущение структуры.

**→ Продуктовый ответ:** Привязка практики к якорям рутины (утро / после тренировки / перед сном), мягкие напоминания, персональная статистика (минуты, сессии). Рекомендации «следующей практики» на основе контекста. Системность через привычку, привязанную к жизни — а не через стрики и геймификацию. Как «чистка зубов», а не как ачивки.

## 4. Структура контента

### 4.1. Типы контента

| Тип | Длительность | Job | Доступ |
|---|---|---|---|
| Quick Reset | 7-10 мин | Job 1 | Free (входная воронка) |
| Focused Session | 15-20 мин | Job 1 + Job 3 | Free / Premium |
| Deep Practice | 25-45 мин | Job 2 | Premium |

Без курсов на старте — чистая библиотека отдельных практик. Курсы можно добавить позже как эволюцию продукта, когда станет понятно, какие практики люди проходят чаще и в какой последовательности.

### 4.2. Категории практик (по функциональному запросу)

Вместо эзотерических названий — функциональные, понятные IT-аудитории:

- **Перезагрузка** — снять напряжение, вернуть ясность (вечерние, после работы)
- **Фокус** — собраться перед важным делом (утренние, перед встречами)
- **Энергия** — поднять тонус без кофеина (дневные, при упадке сил)
- **Сон** — подготовить тело к глубокому сну (перед сном) ← входная точка для нового сегмента
- **Recovery** — восстановление после спорта/тренировки (после физической нагрузки)
- **Глубина** — иммерсивные сессии для внутренней работы (моменты жизненных переломов)

**Контекстные теги (время/ситуация):** между встречами, после работы, перед сном, после тренировки, выходные

**Позиционирование:** «Когда медитировать не получается — дыши.» (реальный инсайт: медитация может усиливать тревогу в стрессе; breathwork — активная практика, работает даже когда голова не отключается)

### 4.3. Карточки практик

Каждая карточка содержит:
- Название (функциональное, не эзотерическое)
- Длительность
- Уровень интенсивности (1-5)
- Тег категории + контекстный тег (время/ситуация)
- «Почему работает» — 1 строка научной базы
- «Что будет происходить» — превью ощущений (снимает страх)
- Краткое описание (1-2 предложения: что будешь делать + какой результат)
- Визуал (уникальный 3D-арт / генеративная графика)
- Техника (название breathing pattern для тех, кто хочет знать)

**Пример карточки:**

## 5. Фичи продукта

### 5.1. MVP (версия 1.0)

**Must have:**
- Каталог практик с фильтрами (категория, длительность, интенсивность, контекст)
- Аудио-плеер с голосовым ведением + музыкой
- Прогресс-трекер (завершённые сессии, привязка к якорям рутины)
- Онбординг: выбор цели (перезагрузка / фокус / сон / энергия / recovery)
- Профиль с базовой статистикой
- На каждой карточке: «Почему работает» (научная микро-база, 1 строка)
- На каждой карточке: «Что будет происходить» (превью ощущений)
- Шеринг практики другу в ТГ (реферальная механика — главный канал роста)
- Профиль проводника (Стас — фото, стиль, подход)

**Nice to have (v1.0):**
- Напоминания с привязкой к якорю рутины (не просто «дыши в 21:00», а «после тренировки» / «перед сном»)
- «Практика для тебя сейчас» (рекомендация по времени суток на главном экране)
- Уровни сложности практик (базовый / продвинутый)

### 5.2. Версия 2.0

- Персонализация: алгоритм рекомендаций на основе истории и времени
- Курсы (серии из 7-30 сессий) — на основе данных о том, какие практики проходят чаще
- Интеграция с HRV-трекерами (Oura, Apple Watch) — показать объективный эффект
- Глубокие курсы с прогрессией техник
- Офлайн-мероприятия: запись на живые сессии через приложение
- Реферальная программа

- 3-5 Quick Reset сессий (полный доступ навсегда)

**Premium (подписка):**
- Полный каталог практик
- Глубокие иммерсивные сессии
- Расширенная статистика
- Новый контент каждую неделю/месяц

**Ценообразование (гипотеза для теста):**

| Настроение | «Бунт с иронией» | «Спокойная сила» |
| UI | Комьюнити-платформа | Инструмент / тулза |

### Визуальные референсы (направление):
- Карточки как на скрине (3D-скульптурные формы, белый/светлый арт на тёмном фоне)
- Oura Ring UI (чистота, метрики, тёмная тема)
- Endel (генеративные soundscapes, минимализм)
- Apple Fitness+ (функциональность + премиальность)

### Тон коммуникации:
- Прямой, без воды — как разговор между коллегами
- Научные факты без занудства — «снижает кортизол», не «активирует парасимпатическую нервную систему блуждающего нерва»
- Допускает глубину без эзотерики — «почувствуй, что происходит в теле», не «открой чакры»
- Уважает время — короткие описания, никаких лонгридов внутри приложения
